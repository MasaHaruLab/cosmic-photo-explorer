# Cosmic Photo Explorer

A photoreal cosmic exploration product for web-first interactive science storytelling.

Current phase: Phase 1 skeleton.

Goals for phase 1:
- Photoreal Milky Way overview
- Low-noise UI
- Clickable anchors
- Cinematic zoom transitions
- Clear boundary between real data and speculative mode

Repo structure:
- `app/` web runtime
- `pipeline/` local asset production helpers
- `data/` manifests and anchor metadata
- `assets/` local placeholder assets
- `docs/` architecture and contracts
- `scripts/prepare_runtime_assets.py` runtime-facing asset scaffold helper
