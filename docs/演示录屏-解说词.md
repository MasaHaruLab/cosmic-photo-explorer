# 星空探索器 · 演示录屏解说词（你自己念）

约 4 分钟，六段，一条"由内向外飞"的线：整片天空 → 拉近 → 跳出银河系 → 飞出太阳系 → 星座真相 → 收尾。
标 `【可删减】` 的地方压一压就是 3 分钟。

**念法说明**
- `/` = 换气 / 短停，念到这里稍微顿一下，不是逗号。
- 英文名后面我都跟了中文。念的时候**中英各念一遍最稳**（评委听得懂、也显专业），嫌绕就只念中文。
- 这是"念给人听"的稿，不是书面语。照着自己平时说话的调子念，别端着。念错一句就停一下重念那半句，后期剪掉，不用从头来。

线上版直接录：`https://masaharulab.github.io/cosmic-photo-explorer/`

---

## 一、开场 · 0:00–0:20

**【画面】** 主页天图**加载完**之后再开始念（别对着"正在加载天图…"讲）。手在星空上拖一下，让整片银河动起来。

**【念白】**
你现在看到的这一整片天，不是画出来的。/ 这是 Gaia（盖亚）——欧洲空间局的一颗卫星，一颗一颗测出来的十几亿颗恒星，它们真实的位置和亮度。/ 我拖一下，整片银河都能自己走。/ 这就是这个项目的起点：我想要一个能真的走进去的星空，底图全是真实数据，不是插画。

---

## 二、拉近 · 0:20–1:05

**【画面】** 点一个锚点（推荐**银心**或**猎户座大星云**）→ 两秒多的电影感俯冲。拉近后底图自动切成 DSS2 照片。再点"双眼对比"，左照片右测量图并排。最后开一下"自动漫游"，让它自己走两站。

**【念白】**
我点银心。/ 两秒多，镜头就像电影一样俯冲下去。/ 拉到足够近，底图会自己从测量图，切成真实的照片——这是 DSS2（数字巡天），上世纪的照相底片扫描出来的深空照片。/ 我还能把两张并排放：左边是照片，右边是同一片天的测量数据，一眼就看出它俩的差别。/ 懒得自己找，就开自动漫游——它像一辆观光车，载着你一站一站看过去。

---

## 三、看图 + 跳出银河系 · 1:05–1:45

**【画面】** 在某个星云上点"在 NASA 图库看这里"，弹出大图灯箱，看两秒关掉。然后点解释层里的**"跳出银河系"**卡片。

**【念白】**
每一个天区，我都接了 NASA（美国航空航天局）的官方照片，点一下，直接在页面里看大图，不用跳走。/ 但我最想让你看的是这一层——跳出银河系。/ Gaia 测了十几亿颗星，听着很多；可把它放进整个银河系，也只是太阳周围的一小片。/ 人类画宇宙地图，才刚开了个头。/ 这句话，我想让每个来的人都撞上一次。

---

## 四、飞出太阳系 · 深空信使 · 1:45–2:55

**【画面】** 顶部导航切到**"深空信使"**页。五条真实航迹 + 日球层顶那个网格球罩先亮个相。点**旅行者一号**的播放键，让它飞：掠木星、掠土星、暗淡蓝点、冲出日球层。飞的同时展开它那张档案卡。
**【可删减】** 压 3 分钟时：只播旅行者一号，档案卡那句略过。

**【念白】**
看完天，接着往外飞。/ 这一页叫深空信使。/ 五艘真的探测器——旅行者一号、二号（Voyager 1、2），先驱者十号、十一号（Pioneer 10、11），还有新视野号（New Horizons）。/ 它们的轨迹不是我随手画的示意图，是 NASA 的 JPL Horizons（喷气推进实验室星历系统）算出来的真实三维轨道。/ 我放旅行者一号飞一遍：掠过木星、土星，回头拍下那张著名的暗淡蓝点，然后一路冲出日球层，飞进星际空间。/ 每一个里程碑，都是真实的日期、真实的位置。/ 点开它的档案卡——1977 年发射，设计寿命五年，现在超期十倍，还在跟地球通信。

---

## 五、星座的真相 · 2:55–3:35

**【画面】** 切到**"星座"**页，选**猎户座**。手动拖动转一下视角，让腰带三星散开、整个猎户人"散架"。

**【念白】**
最后一站，我最喜欢。/ 星座，其实是个障眼法。/ 我把猎户座（Orion）的星，放到它们真实的距离上——你看腰带上这三颗，看着排成一条直线，其实一颗一千多光年、一颗两千光年，差着十万八千里。/ 我一转视角，这个熟悉的猎户人，立刻就散架了。/ 只有从地球这一个方向看过去，它才"恰好"排成那个样子。/ 从小认得的那个猎户座，是站在地球这一个点上，才成立的巧合。

---

## 六、收尾 · 3:35–4:00

**【画面】** 快速掠过**太阳系 / 观星台 / 空间站**几页（各停一两秒）。点右上角**中/EN** 切一下语言，让整站变英文再切回。最后停在首页或"数据与来源"页。
**【可删减】** 压 3 分钟时：省掉掠页，直接切语言 + 收尾。

**【念白】**
这个站还有太阳系、观星台、空间站几页，你可以自己逛。/ 右上角一键，整站就能切成英文，中英双语。/ 我想说的其实就一句：从整片天空，到飞出太阳系，你脚下踩的每一张底图，都是真实测量的数据——不是插画，也不是示意图。/ 方向和品味是我定的，工程我交给 AI，两个人一版一版磨出来的。/ 这就是星空探索器。谢谢。

## 录之前，这几件事做一下（一条过的关键）

1. **先彩排一遍点击路径**，空着念一遍。尤其那个两秒俯冲和自动漫游，手要跟得上嘴。
2. **每一页先打开预热一遍**再正式录——让 Aladin 的瓦片和 NASA 图片都缓存好，录的时候不转圈、不卡。
3. **用线上版录**（上面那个网址），别用本地 dev，稳。
4. 浏览器**全屏、藏掉书签栏、关掉通知**（别让弹窗入镜），窗口调成 16:9。
5. **麦克风离近一点、环境安静**。念错就停一下重念那半句，后期剪掉。
6. 深空信使那页探测器要飞 40 秒左右，**你不用等它飞完**——放起来，讲两句掠木星掠土星，就可以往下走。
7. 别追求零瑕疵。评委看的是这片星空本身，不是你有没有卡壳。

## 万一评委问"这跟 grapeot（鸭哥）的项目什么关系"

照实说，站得住：**灵感来自他公开分享的工作流笔记，但代码是我独立写的**。他做的是 Python 离线渲染管线（把 Gaia 原始数据渲成瓦片）；我用的是 CDS 的公共巡天服务 + Aladin Lite（阿拉丁星图工具）+ 三维轨道数据，技术完全两条路，一行代码没抄，图也没搬。应用内"数据与来源"页里我已经署了名致谢。（我已把两边代码逐项核对过，这个说法是真的。）
